# Vehicle Characteristics Pattern

This ontology defines patterns for vehicle characteristics.

This pattern imports the following files:

- [https://w3id.org/citydata/part1/v1/](https://w3id.org/citydata/part1/v1/)
- [https://w3id.org/itsdata/core/v1/](https://w3id.org/itsdata/core/v1/)

This pattern consists of the following classes:

- [Emission Level Code](EmissionLevelCode.md)
- [Fuel Type Code](FuelTypeCode.md)
- [Goods Type Code](GoodsTypeCode.md)
- [Load Type Code](LoadTypeCode.md)
- [Vehicle Characteristics](VehicleCharacteristics.md)
- [Vehicle Equipment Code](VehicleEquipmentCode.md)
- [Vehicle Type Code](VehicleTypeCode.md)
- [Vehicle Usage Code](VehicleUsageCode.md)
This module defines the following properties:

- [hasFuelType](../properties/hasFuelType.md)
- [hasGoodsType](../properties/hasGoodsType.md)
- [hasGrossMass](../properties/hasGrossMass.md)
- [hasHeaviestAxleMass](../properties/hasHeaviestAxleMass.md)
- [hasHeight](../properties/hasHeight.md)
- [hasKerbMass](../properties/hasKerbMass.md)
- [hasLength](../properties/hasLength.md)
- [hasLoadType](../properties/hasLoadType.md)
- [hasMaximumAllowedMass](../properties/hasMaximumAllowedMass.md)
- [hasNumberOfAxles](../properties/hasNumberOfAxles.md)
- [hasVehicleEquipment](../properties/hasVehicleEquipment.md)
- [hasVehicleType](../properties/hasVehicleType.md)
- [hasVehicleUsage](../properties/hasVehicleUsage.md)
- [hasWidth](../properties/hasWidth.md)
- [yearOfFirstRegistration](../properties/yearOfFirstRegistration.md)


The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](../vehicle-characteristics-pattern.ttl) and the SHACL [restrictions](../vehicle-characteristics-shacl.ttl).
